# 方法

## 1、什么是方法？

方法是封装了代码块的函数，方法可以有返回值，也可以没有返回值。

1. 语法：

   `void`代表没有返回值

   ```java
   // 修饰符 返回值类型 方法名(参数列表) {
   //   方法体;
   //   return 返回值;
   // }

   // public static 是修饰符
   // int 是返回值类型
   // add 是方法名，括号里面的是方法的参数
   public static int add(int a, int b) {
       return a + b;
   }

   // 方法的调用
   add(5,10)
   ```
